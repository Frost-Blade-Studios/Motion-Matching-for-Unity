    Original PDF Documentation

    The original Motion Matching for Unity asset included comprehensive PDF documentation. These resources provide in-depth guides, tutorials, and reference materials.

    Documentation Status

    The PDF documentation below represents the original asset's documentation. While the core concepts remain accurate, some Unity-specific details may differ in Unity 2022+ versions. Cross-reference with the Getting Started Guide for modern workflows.

    Accessing the PDFs

    Option 1: Download from Repository

    All PDFs are included in this repository in the docs/ folder. You can:

    1. Clone the repository
    2. Navigate to the docs/ folder
    3. PDFs are ready to view

    Option 2: View on GitHub

    Visit the GitHub repository and browse the docs/ folder to view or download individual PDFs.

    Recommended Reading Path

    For Beginners

    1. Start here: Getting Started Guide (this site)
    2. Then read: Quick Start PDF
    3. Next: Technical Reference (this site) - Overview sections
    4. Finally: User Manual PDF - as needed for specific features

    For Experienced Users

    1. Review: Technical Reference (this site)
    2. Reference: User Manual PDF for detailed parameters
    3. Optimize: Animation Authoring Guide PDF

    For Animators

    1. Essential: Animation Authoring Guide PDF
    2. Context: Technical Reference - Section 7 (Asset Preparation)
    3. Troubleshooting: FAQ PDF
